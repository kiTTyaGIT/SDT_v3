Данное руководство кратко описывает базовые принципы работы программы. При запуске программы на экране будет краткое описание назначения программы и основные положения по её использованию. Программа работает при вводе данных с двумя языками: русским и английским.
Программа формирует на основе информации, вводимой пользователем, два файла, которые будут храниться в текущем каталоге – файл программ и файл модулей:
?	Файл программ: состоит из записей, каждая из которых включает четыре поля- название программы и три поля названий модулей, входящих в программу (могут быть заполнены не все три поля одновременно);
?	Файл модулей: состоит из записей, каждая из которых включает три поля – названия модуля, названия языка программирования и количества строк исходного текста.
При вводе пользователем служебных символов (/, <, >, *, +, “, . , #, @, %) или пробел/enter в поля, будет выводиться предупреждение об ошибке.
Во время работы на экран выводится меню, в котором пользователю предлагается выбрать одну из двух команд: нажать 1 – для работы с файлом программ, нажать 2 – для работы с файлом модулей; также есть возможность завершить работу с программой, нажав <escape>.
Цифры команд соответствуют такой же цифре на клавиатуре, при нажатии которой происходит соответствующее действие. В случае некорректного ввода команды (ввода буквы, ввода цифры не из перечня, ввода служебного символа) будет выведено сообщение об ошибке и будет дана возможность ввести команду ещё раз (далее это правило соблюдается для всех остальных меню в программе)
При выборе команды пользователю предлагается заполнить каждый из файлов информацией. Данные можно будет заполнять необходимое количество раз. Для завершения работы программы нажать <escape>.
